## Usage

To run the project locally do the following:

- Clone the project into your local machine:</br>
  `git clone https://github.com/Hombre2014/School-library.git`
- Change the working directory to the project's one:</br>
  `cd School-library`
- run the command:</br>
  `ruby main.rb`</br>
- OR simply type</br>
  `./main.rb`
